# AgentView

## Install And Run

```sh
npm ci
```

Run the web app locally:

```sh
npm run dev:all
```

Open `http://127.0.0.1:5173`.

Run the Electron app locally:

```sh
npm run electron:dev
```

Compile and run the packaged Electron app:

```sh
npm run compile:electron
open release/*/AgentView.app
```
